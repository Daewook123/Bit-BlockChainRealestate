## 🏠블록체인기반 부동산 거래 Dapp


 

## ❓ 프로젝트 기본 소개

 #### 프로젝트 배경
- 부동산 거래를 할 때 부동산을 찾아가 공인중개사를 통해 거래를 하곤 한다. 
- 공인중개사를 끼고 거래 할 시 중개 수수료 및 부동산 신뢰도 문제가 발생 할 수 있다. 

 #### 프로젝트 목표
- 블록체인의 특징인 투명성과 보안성을 통한 안전한 거래 
- 판매자와 구매자간 의사소통을 위한 채팅방 개설
- 집 매물 등록 시 해당 집에 대한 NFT 발행
